# 静的変数とは？

**静的変数（static）**

- 「一度だけ初期化されて、プログラムが終わるまでメモリ上に残り続ける」変数。

- **スコープ**：
    - 関数内で `static` を付けると、ブロック外からは見えないが、関数呼び出しをまたいで値を保持する。
    - ファイル（翻訳単位）レベルで `static` を付けると、他のファイルからは参照できないグローバル変数になる。

- **寿命**：プログラム開始時（正確にはロード時）にメモリを確保＆初期化され、プログラム終了時まで存在し続ける。

- **解放タイミング**：
    - 一般的な OS 上の C プログラムでは、プロセスが終了するときに OS がまとめてメモリを回収するため、プログラマが手動で `free` する必要はありません。
    - 組み込みシステムなど OS のバックグラウンドで動かない環境では、リセットや電源断が起こるまでメモリは保持されます。
